  • Сишный контест 873002:
    • simple_page - обязательная, домашняя;
    • segment - необязательная (бонусная) - 1 балл;
    • page - необязательная (бонусная) - 1 балл;
    • segment_page - необязательная (бонусная) - 2 бонусных балла.
    Дело в том, что в 3-м семестре обычно даётся 2 контрольных на бумажках. Одна из задач оттуда - про сегментно-страничную организацию памяти. Её все боятся, пропускают, а ведь она очень простая и даёт много баллов. Так вот, я её оцифровал - segment_page в контесте. (Ну, у меня есть и ещё пару задач из этих контрольных.) В качестве тестов - варианты прошлых лет. Так что, если кто сделает segment_page, у него будет уже в кармане одна контрольная задача следующего семестра. Не упускайте шанс. ;-)
  • Ассемблерный контест 873004:
    • triangle - неравенство треугольника - считаем её семинарской, но кто не успел, делайте дома. Давайте договоримся сделать её с помощью функций на ассемблере. Примерная канва может быть такая: ; parameters: EAX, EBX, ECX ; return: EAX (0 or 1) ; spoil: EDX ; (комментарии к функции обязательны - будем считать их вашим "интерфейсом") check: ; начало функции ... ; проверяем неравенство и печатаем ругательство, если оно нарушается mov eax, 0 ret check_ok: mov eax, 1 ret ; конец функции global _start _start: ; Entry Point ... ; заносим параметры в eax, ebx, ecx... call check cmp eax, 0 ; проверяем результат jz l_exit ; снова заносим параметры в eax, ebx, ecx, но в другом порядке... call check cmp eax, 0 ; проверяем результат jz l_exit ... и т.д.
