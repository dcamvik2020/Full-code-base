• Style Checker v1.11 К началу нового семестра я выпустил новую версию своего Стайл-Чекера, а также для тех, кто ленится скомпилировать из исходников (или потерял доступ к ним), я сделал бинарные сборки под x86, которые теперь будут лежать и обновляться на серверах: Formenos: Linux (x86, x86_64), Windows (x86, x86_64); EJudge: Linux (x86, x86_64), Windows (x86, x86_64). (это два зеркала; симлинк formatter указывает на formatter64, а симлинк formatter.exe на formatter32.exe, т.е. под Linux дефолтная версия 64-битная, а под Windows 32-битная) Использовать довольно просто: ./formatter my_prog.c my_prog_fixed.c После этого my_prog_fixed.c будет гарантированно проходить Coding Style check в контестах (если только, конечно, не содержит синтаксические ошибки компиляции).
