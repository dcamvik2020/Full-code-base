Подключить бинарный модуль к своему проекту в QtCreator/Eclipse/Code::Blocks/VisualStudio можно следующим образом: (в примерах исходный модуль используется brute.c, а бинарный passwd.o(bj), хотя имена могут быть любые)
  • Qt Creator:
    1. Создать простой проект на C без использования библиотек Qt.
    2. Положить passwd.o в папку со своим проектом рядом с main.c.
    3. В файле настроек проекта проект.pro добавить следующую строчку: OBJECTS += passwd.o
    4. Вернуться в Qt Creator, скомпилировать и запустить, как обычно.
  • Eclipse:
    1. Создать обычный C-проект с исходным файлом, например, brute.c.
    2. Положить passwd.o в папку со своим проектом рядом с brute.c.
    3. Вернуться в Eclipse, выбрать слева в окне Project Explorer свой проект, обновить его (нажать F5).
      1. Выбрать пункт главного меню Project -> Properties,
      2. в появившемся диалоге слева выбрать пункт C/C++ Build -> Settings,
      3. в середине на вкладке Tool Settings выбрать пункт MinGW C Linker -> Miscellaneous,
      4. справа внизу в окне Other objects нажать кнопку Add... (зелёненький +) и прописать путь к данному бинарному модулю, например, src\passwd.o
    5. Apply, OK, дальше работаем, как обычно, компилируем, запускаем.
  • Code::Blocks:
    1. Создать обычный C-проект.
    2. Положить passwd.o в папку с проектом, т.е. рядом с .c-файлом.
      1. Затем пункт меню Project -> Build options...
      2. в открывшемся диалоге перейти на закладку Linker settings,
      3. в окне Other linker options написать passwd.o
    4. Всё. ОК, Compile, Run.
  • M$ Visual C++: Как добавить бинарный объектный файл к своему проекту (без Makefile, с помощью среды разработки Visual Studio):
    1. Создаём обычный проект Си с одним исходным файлом.c, который пишете вы.
    2. Кладём файл passwd.obj в папку с проектом, рядом с вашим исходником.c.
      1. Пункт меню Project -> <проект> Properties...,
      2. в открывшемся диалоге слева выбираем пункт Configuration Properties -> Linker -> Input,
      3. а справа в поле Additional Dependencies пишем passwd.obj
    4. Всё, компилируем.
