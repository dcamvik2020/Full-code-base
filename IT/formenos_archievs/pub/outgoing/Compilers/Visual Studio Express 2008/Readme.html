===== УСТАНОВКА =====

  1. Скачиваем дистрибутив: a) Версия 2008 из локальной сети (русский или английский) b) Версии 2012 и более поздние из Интернета.
  2. ISO-шник записываем на диск или монтируем, например, с помощью DaemonTools:
    1. Скачиваем Daemon Tools.
    2. Устанавливаем. Отказываемся от платных лицензий, SPTD, сбора статистик, Яндекс-ерунды.
    3. Монтируем образ.
  3. Устанавливаем Visual C++ 2008. Ничего ей не разрешаем, лишнего (Silverlight, SQL Server) не ставим.
  4. Образ можно отмонтировать, Daemon Tools удалять, если не нужен.

===== ИСПОЛЬЗОВАНИЕ =====

Создание Hello, World:
  1. Запускаем Visual C++ 2008. Начальную страницу можно закрыть.
  2. Создаём проект: Файл -> Создать -> Проект... (File -> New -> Project...) Тип проекта - Общие (General), шаблон - Пустой проект (Empty Project), вводим имя проекта БЕЗ расширения .c или .cpp
  3. Добавляем модуль: Проект -> Добавить новый элемент... (Project -> Add New Item...) Выбираем шаблон "Файл C++ (.cpp)" ("C++ File (.cpp)"), вводим имя исходника (можно насильно указать расширение .c, тогда и программа будет именно Си, а не C++)
  4. Набираем нетленные строчки. После последней значимой строчки лучше добавить переход на новую строку.
  5. СОХРАНИТЬ! Ctrl+S
  6. Скомпилировать F7 !!! ПРОВЕРИТЬ, ЧТО В OUTPUT'Е НЕТ ОШИБОК !!!
  7. Запустить так, чтобы окно не закрывалось после завершения программмы: Отладка -> Запуск без отладки (Debug -> Start Without Debugging) (Ctrl+F5)

----- Отладка -----

Сначала сохранить, затем скомпилировать.
  • F10 (Step Over) - старт отладки, пошаговое выполнение.
  • F11 (Step Into) - пошаговое выполнение, проваливаться внутрь пользовательских функций.
  • Ctrl+F10 (Run to Cursor) - выполнить до строчки, на которой стоит курсор ввода.
  • F5 (Continue) - продолжить выполнение программы до конца или до очередного брекпоинта.
  • F9 (Breakpoint) или клик на левом поле - поставить/снять брекпоинт.
