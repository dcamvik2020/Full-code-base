  1. Скачиваем дистрибутив из этой папки:
    • Для 32-битного Linux
    • Для 64-битного Linux
    • Для 32-битного Windows
  2. Распаковываем в любую удобную папку. ВАЖНО! Полный путь НЕ ДОЛЖЕН СОДЕРЖАТЬ пробелов, русских букв и знаков препинания!

===== ИСПОЛЬЗОВАНИЕ =====

Создание Hello, World:
  1. Запускаем, выбираем, куда будет сохраняться workspace вместе со всеми проектами, страницу Welcome закрываем.
  2. Создаём проект: File -> New -> C Project Тип проекта - Executable -> Hello World ANSI C Project, тулчейн - Linux GCC для Linux или MinGW GCC для Windows, вводим имя проекта БЕЗ расширения .c или .cpp Finish.
  3. Набираем нетленные строчки. После последней значимой строчки лучше добавить переход на новую строку.
  4. СОХРАНИТЬ! Ctrl+S
  5. Скомпилировать - кнопка молотка на панели инструментов. Лучше сразу переключиться на вкладку Console в нижней части экрана.
  6. Запустить - кнопка с зелёной стрелкой на панели инструментов.

----- Отладка -----

  1. Сначала сохранить, затем скомпилировать.
  2. Затем начать отладку - кнопка зелёного жучка на панели инструментов.
  3. Согласиться на переключение на перспективу Debug и поставить галочку "запомнить".
  • F6 (Step Over) - пошаговое выполнение.
  • F5 (Step Into) - пошаговое выполнение, проваливаться внутрь пользовательских функций.
  • Ctrl+R (Run to Line) - выполнить до строчки, на которой стоит курсор ввода.
  • F8 (Continue) - продолжить выполнение программы до конца или до очередного брекпоинта.
  • (Breakpoint) - двойной клик на левом поле - поставить/снять брекпоинт.
