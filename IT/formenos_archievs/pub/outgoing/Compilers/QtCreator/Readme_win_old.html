===== УСТАНОВКА =====

----- Вариант установки с нуля (из Интернета) -----

    0. Если WinXP и не стоит Microsoft Visual C++ 2008 Redistributable, то скачиваем и устанавливаем.
  1. Скачиваем Online installer под Windows.
  2. ВАЖНО! Устанавливать нужно в папку, полный путь которой НЕ СОДЕРЖИТ пробелов, русских букв и знаков препинания!
  3. Устанавливаем нужные компоненты: Тип установки - Custom Проследить, чтобы стояли галочки: Qt SDK -> Development Tools -> Desktop Qt -> Qt 4.8.1 (Desktop) -> Desktop Qt 4.8.1 - MinGW Qt SDK -> Miscellaneous -> Qt Examples и MinGW 4.4 Qt SDK -> Documentation -> Qmake Documentation, Qt Documentation, Qt SDK Documentation, Qt Creator Documentation А остальные (4.8.0, Simulator, Harmattan, Symbian, APIs...) не нужны.

----- Вариант распаковки уже установленной версии (из локальной сети) -----

    0. Microsoft Visual C++ 2008 Redistributable должен быть установлен.
  1. Скачиваем из этой папки Qt_win7.zip
  2. Распаковываем В КОРЕНЬ ДИСКА C:\ т.е. путь к исполняемому файлу среды должен быть: C:\Qt\Tools\QtCreator\bin\qtcreator.exe ВНИМАНИЕ! Путь должен быть ИМЕННО ТАКИМ. Иначе лучше установить с нуля из Интернета.

===== ИСПОЛЬЗОВАНИЕ =====

Создание Hello, World:
  1. Запускаем Qt Creator.
  2. Создаём проект: Файл -> Новый файл или новый проект... При создании проектов надо следить, чтобы они не создавались в папке bin !!! Тип проекта - Другой проект, шаблон - Простой проект на языке C, вводим имя проекта БЕЗ расширения .c или .cpp Далее, далее, Теневую сборку не использовать.
  3. Набираем нетленные строчки. После последней значимой строчки лучше добавить переход на новую строку.
  4. СОХРАНИТЬ! Ctrl+S
  5. Скомпилировать - кнопка молотка слева.
  6. Запустить - кнопка с зелёной стрелкой слева.

----- Отладка -----

  1. Сначала сохранить, затем скомпилировать.
  2. Перед началом отладки надо поставить брекпоинт (F9) куда-нибудь, например, на первую строчку кода.
  • F10 (Step Over) - старт отладки, пошаговое выполнение.
  • F11 (Step Into) - пошаговое выполнение, проваливаться внутрь пользовательских функций.
  • (Run to Cursor) - выполнить до строчки, на которой стоит курсор ввода.
  • F5 (Continue) - продолжить выполнение программы до конца или до очередного брекпоинта.
  • F9 (Breakpoint) или клик на левом поле - поставить/снять брекпоинт.
