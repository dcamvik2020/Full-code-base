===== УСТАНОВКА =====

----- Вариант установки с нуля (из Интернета) -----

  1. Скачиваем Online installer под Windows.
  2. Не надо логиниться и создавать всякие Qt Account, на соответсвующей странице просто нажать Skip.
  3. ВАЖНО! Устанавливать нужно в папку, полный путь которой НЕ СОДЕРЖИТ пробелов, русских букв и знаков препинания! Вариант по умолчанию C:\Qt вполне подойдёт.
  4. Выбираем нужные компоненты (проследить, чтобы стояли галочки): Qt -> Qt 5.6 -> MinGW 4.9.2 32 bit Qt -> Tools -> Qt Creator 4.1.0 Qt -> Tools -> MinGW 4.9.2 (вроде, теперь не обязательно) Либо можно попробовать какую-нибудь другую версию Qt или MinGW. А остальные галочки не нужны.

----- Вариант распаковки уже установленной версии (из локальной сети) -----

    0. Microsoft Visual C++ 2008 Redistributable должен быть установлен.
  1. Скачиваем из этой папки Qt_win7.zip
  2. Распаковываем В КОРЕНЬ ДИСКА C:\ т.е. путь к исполняемому файлу среды должен быть: C:\Qt\Tools\QtCreator\bin\qtcreator.exe ВНИМАНИЕ! Путь должен быть ИМЕННО ТАКИМ. Иначе лучше установить с нуля из Интернета.

===== ИСПОЛЬЗОВАНИЕ =====

Создание Hello, World:
  1. Запускаем Qt Creator.
  2. Создаём проект: Файл -> Создать файл или проект... Тип проекта - Проект без Qt, шаблон - Приложение на языке C, При создании проектов надо следить, чтобы они не создавались в папке bin, и полный путь НЕ ДОЛЖЕН СОДЕРЖАТЬ пробелов, русских букв и знаков препинания!!!! Рекомендуется создать папку C:\Qt\QtProjects и поставить её размещением по умолчанию. Вводим имя проекта БЕЗ расширения .c или .cpp Далее, далее, далее, завершить.
  3. Нетленные строчки уже набраны. После последней значимой строчки лучше добавить переход на новую строку.
  4. СОХРАНИТЬ! Ctrl+S
  5. Скомпилировать - кнопка молотка в левом нижнем углу.
  6. Запустить - кнопка с зелёной стрелкой там же рядом.

----- Отладка -----

  1. Сначала сохранить, затем скомпилировать.
  2. Перед началом отладки надо поставить брекпоинт (F9) куда-нибудь, например, на первую строчку кода.
  • F10 (Step Over) - старт отладки, пошаговое выполнение.
  • F11 (Step Into) - пошаговое выполнение, проваливаться внутрь пользовательских функций.
  • (Run to Cursor) - выполнить до строчки, на которой стоит курсор ввода.
  • F5 (Continue) - продолжить выполнение программы до конца или до очередного брекпоинта.
  • F9 (Breakpoint) или клик на левом поле - поставить/снять брекпоинт.
