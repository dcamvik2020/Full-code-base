===== УСТАНОВКА =====

  1. Скачиваем Online installer под Linux 32 или 64 бит.
  2. Не надо логиниться и создавать всякие Qt Account, на соответсвующей странице просто нажать Skip.
  3. Выбираем нужные компоненты (проследить, чтобы стояли галочки): Qt SDK -> Documentation -> Qmake Documentation, Qt SDK Documentation, Qt Creator Documentation Qt Documentation, Qt SDK -> Development Tools -> Desktop Qt -> Qt 4.8.1 (Desktop) Qt SDK -> Miscellaneous -> Qt Examples А остальные (4.8.0, Simulator, Harmattan, Symbian, APIs...) не нужны.

===== ИСПОЛЬЗОВАНИЕ =====

Создание Hello, World:
  1. Запускаем Qt Creator.
  2. Создаём проект: Файл -> Новый файл или новый проект... Тип проекта - Другой проект, шаблон - Простой проект на языке C, вводим имя проекта БЕЗ расширения .c или .cpp Далее, далее, Теневую сборку не использовать.
  3. Набираем нетленные строчки. После последней значимой строчки лучше добавить переход на новую строку.
  4. СОХРАНИТЬ! Ctrl+S
  5. Скомпилировать - кнопка молотка слева.
  6. Запустить - кнопка с зелёной стрелкой слева.

----- Отладка -----

  1. Сначала сохранить, затем скомпилировать.
  2. Перед началом отладки надо поставить брекпоинт (F9) куда-нибудь, например, на первую строчку кода.
  • F10 (Step Over) - старт отладки, пошаговое выполнение.
  • F11 (Step Into) - пошаговое выполнение, проваливаться внутрь пользовательских функций.
  • (Run to Cursor) - выполнить до строчки, на которой стоит курсор ввода.
  • F5 (Continue) - продолжить выполнение программы до конца или до очередного брекпоинта.
  • F9 (Breakpoint) или клик на левом поле - поставить/снять брекпоинт.
