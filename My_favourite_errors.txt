1. Строки заканчиваются на "\0", поэтому, например, при malloc(чтобы выделить память на
   n-символьную строку) нужно писать:
		 s = (char*) malloc((n + 1) * sizeof(char));
2. Нужно смотреть на ограничения параметров в задаче.
3. Если хотим беззнаковые числа, то их не только объявить надо так, но и считать 
   соответственным образом: 
			unsigned x, y, ... ; // не long long, например(внимательность)
			scanf("%u", ...);
4. Быстрая сортировка в stdlib.h:
			
			int cmp(const void *a, const void *b) {
 			    return *(int*)a - *(int*)b;
 			}// ____comparation
